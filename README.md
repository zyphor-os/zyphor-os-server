# 🧠 Zyphor Server

**Decendance of Zyphor OS**

Zyphor Server is a lightweight server-focused operating system designed for performance, simplicity, and control. Unlike Zyphor Desktop, this version is built specifically for servers and does **not include a graphical user interface (GUI)**.

It is designed for developers, system builders, and learners who want full control of their server environment.
