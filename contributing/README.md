# 🤝 Contributing to Zyphor Server

Welcome!

This is your **first step into the world of open source**.

Zyphor Server is not just a project—it's a learning space where anyone can contribute, even without coding experience.

---

## 🌱 Start Simple

If you're new, you can begin by:

✅ Adding your name to `contributors.md`  
✅ Fixing small typos  
✅ Suggesting ideas  
✅ Asking questions  

No contribution is too small.

---

## 🚀 Your First Contribution

Follow these steps:

1. Fork the repository
2. Go to the `/contributing/contributors.md` file
3. Add your name
4. Commit your changes
5. Create a Pull Request

That’s it—you are now an open-source contributor 🎉

---

## 🧠 Grow With Zyphor

After your first contribution, you can try:

- Editing configurations
- Improving documentation
- Helping fix bugs
- Suggesting features
- Contributing code (when you're ready)

---

## 💡 Philosophy

> Everyone starts somewhere.  
> Even the smallest contribution is a step forward.

Zyphor Server welcomes you—whether you're a beginner or experienced.

---

## ❤️ Final Note

Don’t be afraid to try.  
Don’t be afraid to fail.  

This is a safe place to learn.
